## Job Description: **Machine Learning Engineer**

**Role Summary**
Builds, deploys, and maintains machine learning models in production, converting data and algorithms into reliable, scalable systems that solve real-world problems.

**Key Responsibilities**

* Design, train, evaluate, and deploy ML models.
* Build and maintain data pipelines for training and inference.
* Optimize models for accuracy, performance, and scalability.
* Integrate ML models into production systems and APIs.
* Monitor model performance, drift, and failures.
* Collaborate with data scientists, software engineers, and product teams.
* Perform feature engineering and model selection.
* Ensure reproducibility, versioning, and documentation of models.
* Apply software engineering best practices to ML systems.

**Required Skills**

* Strong proficiency in Python.
* Solid understanding of machine learning algorithms and evaluation metrics.
* Experience with ML frameworks (PyTorch, TensorFlow, scikit-learn).
* Experience with data processing libraries (NumPy, Pandas).
* Knowledge of model deployment and inference pipelines.
* Strong fundamentals in statistics and linear algebra.
* Familiarity with Linux and Git.

**Qualifications**

* Bachelor’s or Master’s degree in Computer Science, Engineering, or related field.
* 2–5+ years of hands-on machine learning experience.
* Experience working on production-grade ML systems.
